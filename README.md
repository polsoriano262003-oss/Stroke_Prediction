# Stroke_Prediction

This project compares Logistic Regression, Random Forest, Gradient Boosting, and Deep Learning models to predict the likelihood of a stroke based on medical and lifestyle features. The focus is on maximizing recall, ensuring individuals at risk are correctly identified.

Features

-Comparative analysis of multiple ML models

-Class imbalance handling (SMOTENC, class weights)

-Data preprocessing and MICE imputation

-Outlier detection using Mahalanobis distance

-Model tuning and evaluation

Models

-Logistic Regression (baseline)

-Random Forest

-Gradient Boosting

-Deep Neural Network

Tech Stack

-Python

-Scikit-learn

-TensorFlow / Keras

-Pandas, NumPy

-FancyImpute (MICE)

-Matplotlib
